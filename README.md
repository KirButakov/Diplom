# Платформа для ведения личного дневника

## Основные функции
Веб-приложение для создания, редактирования и управления записями дневника.

1. Создание и редактирование записей.
Пользователи могут создавать новые записи с заголовком, текстом и изображением.

Возможность редактирования существующих записей.

2. Просмотр записей
Все записи отображаются в виде карточек на главной странице.

Поддержка поиска записей по ключевым словам и тегам.


3. Экспорт записей в PDF
Пользователи могут экспортировать свои записи в PDF-файл для удобного сохранения или печати.


4. Управление тегами
Записи можно помечать тегами для удобной организации и поиска.


5. Тёмная и светлая темы
Пользователи могут переключаться между тёмной и светлой темами для комфортного использования.



7. Социальные функции
Пользователи могут делиться своими записями в социальных сетях (Facebook, Twitter, VK).


8. Уведомления
Пользователи видят уведомления об ошибках при входе или регистрации (например, неверный логин или пароль).


9. Удаление записей
Пользователи могут удалять свои записи с подтверждением действия.

## Технологии
- Backend:

Django (Python)

PostgreSQL (база данных)

- Frontend:

Bootstrap (стилизация)

Font Awesome (иконки)

Google Fonts (шрифты)

- Дополнительные инструменты:

Docker (контейнеризация)

ReportLab (генерация PDF)

## Установка
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/KirButakov/Diplom

2. Настройка переменных окружения

Создайте файл .env в корне проекта и добавьте в него необходимые переменные:


SECRET_KEY=ваш-secret-key

DEBUG=True

DB_NAME=ваш-db-name

DB_USER=ваш-db-user

DB_PASSWORD=ваш-db-password

DB_HOST=db

DB_PORT=5432

3. Запуск с использованием Docker

- docker-compose up --build

4. Применение миграций

- docker-compose exec web python manage.py migrate

5. Запуск сервера

- Приложение будет доступно по адресу: http://localhost:8000.
